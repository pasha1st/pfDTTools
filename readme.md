# pfDTTools - утилита работы с файлами DT - файлами выгрузки из информационных баз 1С
### Внимание! Официально 1С не описывала данный формат файлов, претензии не принимаются.

## Утилита pfDTTools предназначена для распаковки файлов выгрузки информационных баз 1С.

На данный момент это заготовка, для экспериментальных целей.

Утилита предназначена для технических специалистов, например для целей восстановления поврежденных баз данных.
Для компиляции требуется Delphi XE3+. Проверено на Win32/Win64
